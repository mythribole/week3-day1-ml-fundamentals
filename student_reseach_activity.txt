Student Research Activity

1. What is Machine Learning?

Machine Learning is a branch of Artificial Intelligence (AI) that enables computers to learn from data and improve their performance without being explicitly programmed for every task. Instead of following fixed instructions, machine learning systems identify patterns in data and use those patterns to make predictions or decisions. For example, when a student receives movie recommendations on a streaming platform, machine learning algorithms analyse previous viewing history and suggest similar content.

Machine learning works by training a model using historical data. The model learns relationships between inputs and outputs and then applies that knowledge to new data. There are different types of machine learning, including supervised learning, unsupervised learning, and reinforcement learning. Supervised learning uses labelled data, while unsupervised learning discovers hidden patterns without labels.

Machine learning is widely used in modern technology because it can process large amounts of data quickly and accurately. It helps businesses make better decisions, automate tasks, and improve customer experiences. As data continues to grow, machine learning is becoming one of the most important technologies in the world today.

2. What is Linear Regression?

Linear Regression is one of the simplest and most widely used machine learning algorithms. It is used to predict a numerical value by finding the relationship between one or more independent variables and a dependent variable. The goal of linear regression is to draw the best-fit straight line through the data points.

For example, if we want to predict a student's marks based on study hours, linear regression can analyse past data and estimate future marks. The algorithm creates an equation of a straight line:
y = mx + c

where y is the predicted value, x is the input feature, m is the slope, and c is the intercept.

Linear regression is easy to understand and implement, making it a popular choice for beginners. It is commonly used in business forecasting, sales prediction, house price estimation, and performance analysis. Although it is simple, it provides valuable insights into the relationship between variables and serves as the foundation for many advanced machine learning techniques.

3. What are Features?

Features are the input variables or attributes used by a machine learning model to make predictions. They represent the information that helps the model understand patterns in the data. Features are usually stored as columns in a dataset, excluding the target variable that needs to be predicted.

For example, in a student performance dataset, features may include Study Hours, Attendance, and Number of Assignments Completed. These factors influence the final marks of a student. The machine learning model uses these features to learn relationships and make predictions.

Choosing the right features is very important because they directly affect model accuracy. Irrelevant or unnecessary features can reduce performance and increase complexity. Therefore, feature selection is often performed to keep only the most useful attributes.

In real-world applications, features can include age, salary, temperature, product price, customer ratings, and many other types of data. High-quality features help the model learn effectively and produce accurate predictions.

4. What are Labels?

Labels are the output values or target variables that a machine learning model tries to predict. In supervised learning, labels are already known and are used to train the model. The model learns the relationship between features and labels so that it can predict labels for new data.
For example, in a student performance dataset, Study Hours, Attendance, and Assignments may be features, while Marks is the label. During training, the model learns how these features affect the marks obtained by students.

Labels can be numerical values or categories. Numerical labels are used in regression problems, such as predicting house prices or exam scores. Categorical labels are used in classification problems, such as predicting whether an email is spam or not spam.

Without labels, supervised machine learning cannot learn the correct output. Labels act as the answers that guide the model during training. Accurate and reliable labels are essential for building effective machine learning systems and achieving good prediction performance.

5. What is the difference between Training Data and Testing Data?

Training Data and Testing Data are two important parts of a machine learning dataset. The dataset is usually divided into these parts so that the model can be trained and evaluated properly.

Training Data is used to teach the machine learning model. During training, the model learns patterns, relationships, and trends from the data. The larger the training dataset, the better the model can understand the problem. Usually, about 70% to 80% of the data is used for training.
Testing Data is used after training to evaluate the model's performance. The model makes predictions on testing data that it has never seen before. This helps determine whether the model can generalize to new situations.

The main difference is that training data is used for learning, while testing data is used for evaluation. Using separate testing data prevents overfitting and provides a fair measure of model accuracy. Both datasets are essential for building reliable machine learning model

6. Where is Machine Learning used in real life?

Machine Learning is used in many real-life applications across different industries. One common example is recommendation systems used by streaming platforms and online shopping websites. These systems analyze user behavior and suggest movies, songs, or products that match personal preferences.

In healthcare, machine learning helps doctors diagnose diseases, analyze medical images, and predict health risks. In finance, it is used for fraud detection, credit scoring, and stock market analysis. Banks use machine learning to identify suspicious transactions and improve security.
Machine learning is also widely used in transportation. Self-driving cars use machine learning algorithms to recognize roads, traffic signs, and obstacles. Navigation apps use it to predict traffic conditions and suggest faster routes.

Other applications include voice assistants, facial recognition systems, spam email filtering, customer support chatbots, and predictive maintenance in industries. As technology advances, machine learning continues to transform everyday life by making systems smarter, faster, and more efficient.
