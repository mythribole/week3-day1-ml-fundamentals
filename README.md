# Machine Learning Fundamentals Challenge

## Objective

The objective of this project is to understand the fundamentals of Machine Learning using Scikit-learn. This assignment covers the complete machine learning workflow including dataset exploration, feature selection, train-test split, model training, evaluation, prediction, and data visualization.

Features:

* Study_Hours
* Attendance
* Assignments

* Marks
